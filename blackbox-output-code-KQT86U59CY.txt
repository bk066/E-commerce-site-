# Household Utility Store - Full E-Commerce Website

A complete, self-contained e-commerce website for a household utility store with backend API, SQLite database, and responsive frontend.

- **Product Catalog**: Browse groceries and cleaning products with grid layout.

- **Responsive Design**: Mobile-first UI with specified color scheme.

## Tech Stack
- **Backend**: Node.js + Express + SQLite3 + bcrypt + jsonwebtoken
- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript (Fetch API)
- **Database**: SQLite (local, no external dependencies)
